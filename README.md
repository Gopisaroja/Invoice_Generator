# 🧾 Python Invoice Generator

A Python-based PDF Invoice Generator built using **Python** and **ReportLab**. This project generates clean and professional invoices with automatic GST calculation, invoice numbering, customer details, and a business-ready layout.

## ✨ Features

* Generate professional PDF invoices
* Automatic GST (20%) calculation
* Invoice numbering (e.g., INV-100)
* Customer and business information
* Quantity, price, and total calculation
* Professional invoice layout
* Authorized signature section
* Footer with branding

## 🛠️ Technologies Used

* Python 3
* ReportLab

## 🚀 Installation

1. Clone this repository

```bash
git clone https://github.com/your-username/python-invoice-generator.git
```

2. Navigate to the project folder

```bash
cd python-invoice-generator
```

3. Install the required package

```bash
pip install -r requirements.txt
```

4. Run the program

```bash
python main.py
```
